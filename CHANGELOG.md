# Changelog

All notable changes to `filament-inbox` will be documented in this file.

## v1.0.0 - 2026-04-24

### Added

- Inbox, Sent, Starred, and Trash pages
- Message threading with Reply and Reply All
- Message forwarding with original message header
- Read receipts with per-recipient tracking
- Star/unstar messages
- Rich text editor with file attachments
- Bulk actions (mark as read, move to trash)
- Search and filter (by sender, unread, starred)
- Database notifications via Filament
- Inbox stats widget (unread, total, starred)
- Dark mode support
- Multi-tenancy support (Filament native)
- Configurable user model
- Laravel event system (MessageSent, MessageRead, MessageStarred, MessageTrashed, MessageRestored, MessageForwarded)
- Test factories for Message and MessageRecipient
- Translations: English, Japanese, Spanish, French, Portuguese (Brazil), Chinese (Simplified), Korean, German
